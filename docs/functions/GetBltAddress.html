GetBltAddress

Availability All terminals
Description This function returns the current remote Bluetooth® address that's stored in flash memory.
Syntax unsigned char* GetBltAddress( void );
Arguments None
Returns The Bluetooth® address that is stored in the flash memory.
Remarks The SetBltAddress() function is obsolete, because with BLE the remote Bluetooth® device connects to your terminal and not the other way around
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    for(;;)
    {
        printf("\Blt address is: %s", GetBltAddress());
        ResetKey();
        while( !kbhit() )
            Idle();
    }
}
