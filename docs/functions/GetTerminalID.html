GetTerminalId

Availability All terminals
Description This function returns the number that is set in the system menu, under the "Set Terminal ID" option. It is also possible to use the function SetTerminalId() to change the value of the terminal ID.
Syntax unsigned int GetTerminalId( void );
Arguments None
Returns The number set in the system menu.
Remarks The method to activate the system menu for each device can be read about here.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    SetTerminalId( 1234 );

    for(;;)
    {
        ResetKey();

        while(!kbhit())
            Idle();

        printf("\nTerminal ID: %06d", GetTerminalId());
        Sound(TSTANDARD, VHIGH, SHIGH, 0);
    }
}
