PutBuffer

Availability All terminals
Description PutBuffer() transmits len number of characters of string through the serial communications port that was last opened by ComOpen().
Syntax int PutBuffer( const unsigned char *string, unsigned int len );
Arguments
const unsigned char *string
string is the buffer containing characters to be transmitted.
unsigned int len
len is the number of characters to be transmitted.
Returns OK on success or ERROR on failure.
Remarks - When no port is currently open, ERROR will be returned.
- Use the function PutString if you want to transmit the complete string
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    unsigned char packet[] = "22-\xFF\xFF";   //SYN,SYN,ENQ,PAD,PAD
    
   ComOpen( COM0 );    // Open the COM port for USB-VCP

    PutBuffer(packet, 5);

    for(;;)
    {
        Idle();    // Very important to lower the power consumption
    }
}
