BltUnpair

Availability All terminals
Description Removes all of the Bluetooth pairings.
Syntax void BltUnpair( void );
Arguments None
Returns None
Remarks This function removes all paired (bonded) devices and their link keys from flash memory.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    BltUnpair();
    for(;;)
    {
        Idle();
    }
}
