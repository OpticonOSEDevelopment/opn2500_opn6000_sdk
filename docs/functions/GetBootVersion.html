GetBootVersion

Availability All terminals
Description This function returns the version of the Bootloader of the handheld terminal. The version is returned in an 8 character long string, and is terminated by the null character '\0'. The first and third character of the version defines the terminal type, the second character defines the software type (in this case A)
Code Terminal
FAN OPN2500
FAM OPN6000

Examples of version strings are:
FANV0030 = OPN2500 Bootloader. version 0.30
FAMV0030 = OPN6000 Bootloader. version 0.30
Syntax char *GetBootVersion( void );
Arguments None
Returns A pointer to a zero terminated string with the version number.
Remarks None
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    int key;

    ResetKey();
    printf("\nPress trigger key");
    for(;;)
    {
        if( (key = getchar()) != EOF)
        {
            ResetKey();
            switch( key )
            {
                case TRIGGER_KEY:
                    Sound( TSTANDARD, VHIGH, SHIGH, 0);
                    printf("\Bootloader = %s", GetBootVersion());
                    printf("\nO/S = %s", GetOsVersion());
                    break;
                default:
                    Sound( TCLICK, VHIGH, SLOW, 0);
                    break;
            }
        }
        Idle();
    }
}
