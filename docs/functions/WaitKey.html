WaitKey

Availability All terminals
Description This function blocks execution until a key is pressed.
Syntax void WaitKey( void );
Arguments None
Returns When a key is pressed, WaitKey() will return and program execution will resume.
Remarks Some variants of this function return an integer, which represents the key pressed.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    int color = GREEN;

    for(;;)
    {
         printf("\nPress any key");

         WaitKey();

         ResetKey();

         GoodReadLed(color, 50);        // ON, 1 second

         color = (color == GREEN) ? (RED) : (GREEN);
    }
}
