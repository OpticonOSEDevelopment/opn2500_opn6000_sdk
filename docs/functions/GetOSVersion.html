GetOsVersion

Availability All terminals
Description This function returns a pointer to the version of the currently loaded operating system. The pointer that is returned, points to a zero terminated character string. The first and third character of the version string defines the terminal type, the second character defines the software type (in this case B, operating system software).
Code Terminal
FBN OPN2500
FBM OPN6000

Examples of version strings are:
FBNV0128 = OPN2500 O/S. version 1.28
FBMV0128 = OPN6000 O/S. version 1.28
Syntax char *GetOsVersion( void );
Arguments None
Returns A pointer to a zero terminated string with the version number.
Remarks None
Example 
#include <stdio.h>
#include "lib.h"

#define VERSION "3894.1"               //Application version number


void main( void )
{
    int key;
    
    printf("\nPress trigger key");
    for(;;)
    {
        ResetKey();
        if( (key = getchar()) != EOF)
        {          
            switch( key )
            {
                case TRIGGER_KEY:
                    Sound( TSTANDARD, VHIGH, SHIGH, 0);                                       
                    printf("\nO/S = %s", GetOsVersion());
                    printf("\nAPPL = %s", GetApplVersion);
                    break;
                default:
                    Sound( TCLICK, VHIGH, SLOW, 0);
                    break;
            }
        }
        Idle();
    }
}
