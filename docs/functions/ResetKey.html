ResetKey

Availability All terminals
Description This function clears the keyboard buffer.
Syntax void ResetKey( void );
Arguments None
Returns None
Remarks This function is useful when one wants to wait for a key press. Call this function to prevent a previous keystroke, still in the keyboard buffer, causing e.g. kbhit() to return with TRUE.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    int x, y;

    for(;;)
    {
        for( x=0; x < 96; x++)
            drawline( x, 0, 50, 24, 1);

        for( y=0; y < 48; y++)
            drawline( 96, y, 50, 24, 1);

        for( x=96; x >= 0; x--)
            drawline( x, 47, 50, 24, 1);

        for( y=47; y >= 0; y--)
            drawline( 0, y, 50, 24, 1);

        printf("\nPress a key");
        ResetKey();
        while(!kbhit())
            Idle();
    }
}
