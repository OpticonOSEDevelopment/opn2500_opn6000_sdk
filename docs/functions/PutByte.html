PutByte

Availability All terminals
Description PutByte() transmits a byte through the serial communications port that was last opened by ComOpen().
Syntax void PutByte( unsigned char c );
Arguments
unsigned char c
c is the byte that is to be transmitted.
Returns None
Remarks - Note that on the default port (COM9) is automatically opened at startup.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    unsigned char ch;

    ComOpen( COM9 );    // Open the COM port for USB-VCP

    for(;;)
    {
        if( (ch = GetCom( 0 )) != -1)
        {
            PutCom( ch );
            PutByte( ch );
        }
        Idle();    // Very important to lower the power consumption
    }
}
