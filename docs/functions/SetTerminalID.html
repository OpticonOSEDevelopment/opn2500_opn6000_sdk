SetTerminalId

Availability All terminals
Description SetTerminalId sets the ID of the terminal. This can also be done in the system menu, but this function can be used to set the ID in the application. The ID can be read with the function GetTerminalId().
Syntax void SetTerminalId( unsigned int id );
Arguments
unsigned int id
The new value for the terminal ID. The value must be smaller then 10000.
Returns None
Remarks To read the terminal ID, use the function GetTerminalId(). Passing a value of 10000 or larger will result in undefined behavior.

Note that the terminal ID does not necessarily correspond to the term_id argument in neto_receive() and neto_transmit(). It is merely meant for your own identification purposes.

For a description of how to activate the system menu, click here.
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    SetTerminalId( 1234 );

    printf("\nterminal ID\n%d", GetTerminalId() );

    for(;;)
        Idle();
}
