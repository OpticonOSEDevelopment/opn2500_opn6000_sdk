DelayMs

Availability All terminals
Description This function pauses execution for a number of milliseconds, specified by the argument ms.
Syntax void DelayMs( unsigned short ms );
Arguments
unsigned short ms
The number of milliseconds to wait.
Returns None
Example 
#include <stdio.h>
#include "lib.h"

void main( void )
{
    for(;;)
    {
        Sound(TSTANDARD, VHIGH, SHIGH, 0);
        DelayMs( 500 );    // sounds a beep every 500 milliseconds 
    }
}
