Introduction

This SDK describes how to write C language applications for the OPN-2500 & OPN-6000.

The purpose of this SDK is to make the development of custom applications quick and easy.

Below a list of the main topics that are described in this manual is shown.

1. Installation of the development system
Descibes the installation of this SDK, drivers and editing tools.

2. Application development
This section describes how to develop, port or customize an application.

3. Functions
In this section, the Operating system and relevant ANSI C functions that are available to the application developer are described.

4. Snippets
Some commonly used functions, like database and input functions, are placed in this snippets-chapter. All functions are accompanied with a small example.

5. Loading software on the terminals
On all terminals runs an OS (operating system) and an application. How the OS and the application can be loaded onto each type of terminal is described in this section.

6. System settings
The SystemSetting()-function makes it possible to change a lot of aspects of the reading behaviour of handheld terminals and their communication settings. In this chapter the complete list of available system settting options are described of each type of terminal.
Note: All options can also be found on OptiConfigure

7. Tools
In this chapter some tools are described that can be use to load software, communicate with your device or develop PC/mobile applications.

8. Support
In this chapter contact information is given to get additional user support.
