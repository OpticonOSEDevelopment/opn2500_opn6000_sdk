Installation of the development system

1. System requirements

The prerequisites for using this SDK are:
  • Latest SDK files from this repository
  • SEGGER Embedded Studio for ARM (download from here: SEGGER Embedded Studio)
  • Appload firmware upload application for Windows (download from here: Appload)
  • Opticon USB Drivers (download from here: Opticon USB Driver)

2. License

The SEGGER Embedded studio can be licensed for free when using devices with a Nordic CPU like the OPN-6000 / OPN-2500.
For more info see: Get License for Nordic Semiconductor Devices
It can be useful to attach an nRF52 DK board to your PC, so the license is automatically recognized, although not it's not mandatory to build applications

3. Installation

  • Download and extract this complete repository into a working folder
  • Download and install the latest Opticon USB Driver)
  • Download and install the latest Appload version
  • Install the SEGGER Embedded Studio for ARM
