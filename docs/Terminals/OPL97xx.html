Loading software on the OPL97xx

Image
On all OPL97xx terminals runs an OS (operating system) and an application. The OS and the application can be loaded by a small PC application called APPLOAD.
Software codes The different models of the OPL97xx terminals have their own Operating System version that can be distinguished by an unique 3 letter code (starting with 'LB'). Before you try to load a new Operating System on an OPL97xx you have to know which Operating System file (i.e. LBBVxxxx.S32 for the OPL9723) must be loaded on your terminal. In the table below you can find which software code belongs to which OPL97xx terminal.

Code Terminal
LBB OPL9723, OPL9725, OPL9726, OPL9727
LBC OPL9724 (old*)
LBD OPL9728
LBF OPL9723R (Also known as the OPL9738)
LBO OPL9712
LBP OPL9713
LBR OPL9724 (new*)

*Note: The OPL9724s that have been produced till approximately 2006 have another software code than the newer OPL9724s. To determine the software code of an OPL9724 with certainty, you can find the version of the Operating system (OS) in the System menu by selecting Show version.
