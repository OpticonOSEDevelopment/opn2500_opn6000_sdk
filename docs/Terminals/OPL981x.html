Loading software on the OPL981x

Image
On all OPL981x terminals runs an OS (operating system) and an application. The OS and the application can be loaded by a small PC application called APPLOAD.
Software codes The different models of the OPL981x terminals have their own Operating System version that can be distinguished by an unique 3 letter code (starting with 'LB'). Before you try to load a new Operating System on an OPL981x you have to know which Operating System file (i.e. LBBVxxxx.S32 for the OPL9813) must be loaded on your terminal. In the table below you can find which software code belongs to which OPL981x terminal.
