Loading software on the PX35 and PX36

Image
The PX35 and PX36 terminals run an OS (operating system) and an application. The OS and the application can be loaded by a small PC application called APPLOAD.
Software codes The PX35 and PX36 terminal uses an Operating System version that contains a unique 3 letter code. Before you try to load a new Operating System on a PX35 or PX36 terminal you have to know which Operating System file is meant for your terminal.
In the table below you can find the software code for the PX35 and PX36 terminals.

Terminal Code
PX35 XBF
PX36 XBT
