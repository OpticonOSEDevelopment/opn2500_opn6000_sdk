Loading software on the OPH100x, OPH300x and H13

Image
All H13 and OPH100x terminals run an OS (operating system) and an application. The OS and the application can be loaded by a small PC application called APPLOAD.
Software codes The H13 & OPH100x terminals use the an Operating System version that contains a unique 3 letter code (starting with 'XB'). Before you try to load a new Operating System on a H13 or OPH100x terminal you have to know which Operating System file (i.e. XBAVxxxx.S2) must be loaded on your terminal. In the table below you can find the software code for the OPH100x terminals.

Terminal Code
OPH1004 XBC
OPH1005 XBP
OPH3000 XBM
H13 XBC
OPH1004 To load a new OS or application onto the OPH1004 or OPH1005, you need a CRD1001 cradle, set to auto, or 115K2 baudrate.
Load OS or application.

IrDA or Cradle

Make sure that the terminal is turned OFF.
Press [.]+[1]+[power] keys at the same time, to activate the system menu.
Press the [Down] key once, to select Download appl.
Press the [Ent] key.
Place the terminal in the cradle or in front of the IrDA adapter.
Select the OS or application file in appload.
OPH3000 To load a new OS or application onto the OPH3000, you need a usb cable.
Load OS or application.

mini-USB cable

Make sure that the terminal is turned OFF.
Press [Fn1]+[BS]+[power] keys at the same time, to activate the system menu.
Press the [Down] key twice, to select Downl. Software
Press the [Ent] key.
Press the [Down] key once, to select USB
Press the [Ent] key.
Connect the mini-USB cable to the terminal
Select the OS or application file in appload.
