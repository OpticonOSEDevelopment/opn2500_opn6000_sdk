Character maps and key codes

This section of this help file contains more detailed information about the special key codes and character maps of the different types of terminals.

Key codes are returned by functions like getchar() when the user presses a key. Press the link below to view the special key codes of each type of terminal.

View the special key codes

The character map of a terminal contains the graphical information of all the characters that can be displayed on the screen of the terminal. Press the link below to view the character maps of the different types of terminals.

View the character maps

All handheld terminals support a list of escape sequences that can be used in functions like printf() or putc() to denote special characters like a form feed, backspace and inverted text display. Press the link below to view the list of special characters.

View the special characters
