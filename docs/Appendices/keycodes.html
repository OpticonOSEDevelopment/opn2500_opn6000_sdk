Special key codes

The key codes for special keys are defined in the lists below. The key codes are returned by functions like getchar() when the user presses a special key.
Note that not all these keys are available on all terminals.

Key definition  Description Remarks
BS_KEY Backspace key
ENT_KEY Enter key
CLR_KEY Escape key
TRIGGER_KEY Trigger key
Q1_KEY Option key 1 Is the same as UP_KEY
Q2_KEY Option key 2 Is the same as DOWN_KEY
UP_KEY Up arrow key Is the same as Q1_KEY
On the OPN200x, RS3000 and PX20, UP_KEY and DOWN_KEY are the same: The small key.
On the OPL9723, OPL9724 and OPL9727, this key has a small '1' marking.
DOWN_KEY Down arrow key Is the same as Q2_KEY
On the OPN200x, RS3000 and PX20, the UP_KEY and DOWN_KEY are the same: The small key.
On the OPL9723, OPL9724 and OPL9727, this key has a small '2' marking.
LEFT_KEY Left arrow key Is 'shift' plus UP_KEY
RIGHT_KEY Right arrow key Is 'shift' plus DOWN_KEY
F1_KEY Function key 1
F2_KEY Function key 2
F3_KEY Function key 3
F4_KEY Function key 4
F5_KEY Function key 5
F6_KEY Function key 6
F7_KEY Function key 7
F8_KEY Function key 8
BL_KEY Backlight key
SHIFT_KEY Shift key Only when shift is handled by the application, which is enabled by using SystemSetting("*R");
PWR_KEY Power key Only when power is handled by the application, which is enabled by using SystemSetting("*P");
