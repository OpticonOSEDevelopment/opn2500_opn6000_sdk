System settings

All terminals and bar code readers of Opticon can be configured using the options of 2 to 4 characters listed on OptiConfigure.
While bar code readers can only be configured by reading the menu labels of these options or by serial commands, the OPN-2500 and OPN-6000 can also be configured by passing the appropriate menu options to the SystemSetting()-function.

On OptiConfigure all options that are available for OPN-2500 and OPN-6000 can be found. Click on the appropriate links below to view all available menu options for each type of terminal.
