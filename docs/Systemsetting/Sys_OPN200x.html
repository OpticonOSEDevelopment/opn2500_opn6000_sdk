System settings of the OPN200x, RS3000 and PX20

Below a list of all menu options that can be used in the SystemSetting() function of the OPN200x and PX20 are shown.
The settings depicted in bold are default values.
This list is updated regularly. Please contact your local Opticon dealer if your application needs a setting that is not in the list to find out whether it is already implemented or not.

Note: PDF417 and Codablock-f decoders are no longer supported on OPN2004/5/6 OS version V0048 and later
