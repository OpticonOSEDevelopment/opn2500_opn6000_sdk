Tools

The following tools can be of use during the development applications and upgrading of terminals

Tool Description
Appload Loads operating systems (OS) and applications on terminals, barcode readers (and cradles)
OseComm32 Transfer files from and to Opticon Handheld terminals using the OseComm protocol
OPN Companion Application Tool for managing your OPN-2500 and OPN-6000 in OPN-2001 simulation mode (default firmware only)
