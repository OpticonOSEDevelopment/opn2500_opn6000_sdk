Bmp2H

Description
Convert a bitmaps to a header image format.
Terminals The terminals that support image header files are the ones that support the showpicture() function. The convert file is a header file that contains a const array that holds the screen image.
Options Description of the available options:
File Description
Open Open a bmp file you would like to convert to a header image format.
Save As Converts the loaded bmp file to the header image.
Exit Quit the bmp2h application.

Options Description
Display size Choose the matching screen size.
Initial directory Initial directory to open BMP file.

About Description
About Shows the version number of the Bmp2H file.
Conversion The bitmap image before conversion.

The header file after the bitmap screen.bmp is converted.
