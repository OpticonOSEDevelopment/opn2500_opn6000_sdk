Options
Files
This box holds the files that are to be sent to the handheld terminal. To add a new file, click the Add button below the box. A file dialog appears, select one or more files and click the Open button the selected files will now appear in the box. To remove one or more files from the list, mark the file(s) and click on the Remove button. The clear the whole list click on the Clear button.
