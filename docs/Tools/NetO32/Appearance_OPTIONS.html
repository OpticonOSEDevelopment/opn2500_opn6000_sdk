Appearance Options

Description Press on the different areas of the screenshot below to browse through this NetO32 help.
Options
Start application in tray
Enabling the Start application in tray minimizes the NetO32 application to the system tray when the application is executed. Default this option is not enabled.

Show progress message box when minimized
Enabling the show progress message box when minimized, show a message window when the main NetO32 application is minimized to the tray. The message box will popup when the NetO protocol is transferring data. Default this option is enabled.

Show file progress
Enabling this options will show a file progress bar in the progress message window. This option has only effect when the Show progress message box when minimized is enabled. Default this option is enabled.

Show total progress
Enabling this options will show a total file progress bar in the progress message window. This option has only effect when the Show progress message box when minimized is enabled. Default this option is enabled.

Show progress info
Enabling this options will show the progress status messages in the progress message window. This option has only effect when the Show progress message box when minimized is enabled. Default this option is enabled.

Run protocol on start
When this option is enabled and the NetO32 application is executed the NetO protocol is directly started. Default this option is not enabled.
