Options
Receive Directory
Receiving directory displays the directory into which the data files will be stored that are received from the handheld terminal(s). To change the directory press the small folder button. A directory selector dialog appears. After selecting the correct directory click the OK button to keep the settings.


Downloaded files
There are three options for the file(s) that have been downloaded from the handheld terminal(s). The default value is 'Append'.

Option Description
Append Append the received file to the contents of the same filename that already exists. When the file does not exist a new file is created.
Overwrite Overwrites the old file with the same name if possible, a previous downloaded data file is lost. If the downloaded file does not exist then a new file is created.
Unique Create a new filename every download. When the a file with the same name already exists, the extension of the filename is changed. The possible available values for the extension range from 000 to 999.
